# 自动批量选择特定的radio按钮
```javascript
STR_XPATH=".//input[@type=\"radio\" and @value=\"b\"]";
var result = document.evaluate(STR_XPATH, document, null, XPathResult.ANY_TYPE, null);
var i=1;
while (nodes = result.iterateNext()) {
	nodes.click();
	i++;
}
```
